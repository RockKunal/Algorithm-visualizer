Some Instructions For Visualizing it

Yellow Color means array elements . When the the color becomes orange means it is the minimum element found so far. At each pass (iteration) the element is started with greenyellow color. LighBlue Color means that the current element is being checked with the element found so far(which is in orange color.) After that the minimum element is swapped in the begining.

* Entering array is an mandatory field

Add Array Elemnts separated by space, Space will be considered as spliting point
