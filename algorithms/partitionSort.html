QUICK SORT

Some Instructions For Visualizing it

Quick Sort is based on divide and Conquer.We select a ppivot element and from left we find element greater than pivot element and if we found we swap it and from right we find smaller element than pivot .If found again we swap. When the array will be displayed it will be in yellow color. Darkorange color means the pivot element. If element is in coral color means that we looking for an element greater than pivot. Pink color means we are looking for an elemnt smaller than element .If we find such elements we swap them. After one pass the pivot selected will be at its correct position and from that index array is partitioned into two which is showed by different color. Recursively we solve both parts with same alogorithm.
